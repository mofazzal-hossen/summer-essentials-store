# ☀️ SunCart - Summer Essentials Store

SunCart is a modern, premium summer eCommerce platform designed to provide a seamless shopping experience for seasonal products like sunglasses, outfits, skincare, and beach accessories.

## 🚀 Live URL
[https://summer-essentials-store-six.vercel.app/](https://summer-essentials-store-six.vercel.app/)

## 🌟 Key Features
- **Modern Premium Design**: Unique aesthetic with glassmorphism and smooth animations.
- **Dynamic Product Browsing**: Explore summer essentials with category filtering.
- **Product Details**: Detailed view for every product with related recommendations.
- **Secure Authentication**: Fully integrated with **Better Auth** (Email/Password & Google Login).
- **Protected Routes**: Product details and profiles are secured and require login.
- **User Profile**: View and update your profile information (Name, Photo URL).
- **Responsive Layout**: Optimized for mobile, tablet, and desktop devices.
- **Summer Tips & Brands**: Curated extra sections for a complete summer vibe.

## 🛠️ Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS 4, DaisyUI
- **Components**: HeroUI (formerly NextUI)
- **Authentication**: Better Auth
- **Database**: Better SQLite3 (Local)
- **Icons**: Lucide React, React Icons
- **Animations**: Framer Motion, Animate.css (Utility classes)

## 📦 NPM Packages Used

## 🛠️ Setup & Local Development
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Create a `.env` file with `BETTER_AUTH_SECRET`.
4. Run the dev server: `npm run dev`.
